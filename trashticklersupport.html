Trash Tickler Support

  
 






                      Q & A

Included below are some questions and answers regarding Trash Tickler. Please take a look at these before posting a support request as your question may have already been addressed. Thanks for your help.

Overall

Q: What is the current version?
A: 1.61

Calendar


Q: How do I move between months or years on the calendar?
A: Swipe your finger up, down, left or right to move a month or a year at a time. Pinch with two fingers to return to the current month. There are visual controls below the calendar that can be tapped to perform the same functions.

Q: How do I return to the current day when the calendar displays a past or future month?
A: Tap the small calendar month icon below the larger calendar month or pinch the screen with two fingers.

Q: How do I setup the calendar?
A: Tap the configuration icon in the upper left-hand corner on the calendar page. This will bring up the configuration options for services, holidays and reminders.

Q: How do I return to the calendar page from the configuration options page?
A: There is a small calendar month icon in the upper left-hand corner of the configuration options page. Tap it to return to the calendar page.
 
Q: What is that red number that floats into place on the calendar?
A: The red number is the current day. It only appears in the current month. It will advance as the year progresses, disappear when the calendar is in a future or past month and reappear when you return to the current month.

Q: What are the green numbers on the calendar?
A: Those are the dates configured as holidays. They are used to adjust the calendar to coincide with the provider schedule. You can add them on the configuration options page via the green plus button. On the iPad the green plus button is about midway down the configuration page. On the iPhone or iPod touch it's located at the top right-corner of the Holidays page.

Services

Q: How many services can be configured? What is their purpose?
A: There are three collection services. Refuse, Recycling and Landscape.

 Q: Each service provides seasonal date ranges but can I block out certain weeks? For example I want to alternate weeks.
A: That can be done with one or both of the following options. 

The first option is through the configuration of multiple service collection templates. Within the application you can specify the months and weeks in which a service collection day (or days) will appear. Depending on the needs multiple templates might be required. The templates apply to all years - based on the seasonal date ranges. More details on the seasonal date ranges can be found in the application itself. As a suggestion configure a few templates on your own. Then review the calendar and alter the configuration as necessary. If all else fails you can also send us a note on what you are trying to setup and we can take a look.

The second option is through the use of a holiday with a weekly handling code. This option is more useful for short periods of time while the first option is better for longer periods. This option as a result might be more tedious to configure if many weeks are involved. For more details on the holiday option see the "Holidays" section below.

Holidays

 
Q: How are holidays displayed on the calendar?
A: The day on the calendar will be shown in green.

Q: Do any of the handling codes fully block a green indication on the calendar?
A: No, to avoid confusion any defined holiday regardless of handling code will display as green on the calendar. If the handling code intends to remove a service you'll see the service for the holiday altered accordingly.

The only item that would partially block a green indication is if the current day of the year is also defined as a holiday. In this case you'll see green fully until the red indication for today settles on top. The green indication will remain as a left shadow. 

Q: How do I prevent defined services for a week or more on the calendar?
A: There are two options. The first was explained earlier in the "Services" section above. The second option is to enter a holiday with a handling code that cancels services through the end of the week. All services within this time period (holiday date through the week ending on Saturday) will be cleared from the calendar and no reminders will be generated for that period.

Q: How do I move services ahead a day for a selected day or week?
A: Define a holiday with a handling code that reschedules the services one day later either for the holiday alone or starting on the holiday through the end of the week.

Q: One of my holidays is missing from the calendar. Any suggestions?
A: Check if it is listed in the configured holidays table. If you see it there please check to see what month, day and year it's actually scheduled to appear. New holidays default to the month and year currently displayed on the calendar if no change is made. As a result it might be possible to unintentionally create a holiday in the past or future. Then you might not see it on the calendar without swiping to the month/year in which it's configured.  If the date is not what you need you can delete and re-add the date without looking through the calendar itself.

Note also - if the holiday falls on the current day the calendar will show the day as red with a slight green shadow to the left. If the green shadow is not apparent you can swipe away from the current month and then pinch back. This will allow a full view of what is below today before the float animation completes.

Q: I cannot add a holiday date due to date duplication. What should I do?
A: You may only add a date once to the holiday list. Dates with a handling code indicating "...to the end of the week" actually include a range of dates as well which may be causing the duplication. When in doubt check the overall holiday list. It's sorted in date order. If a range of dates applies you'll see the start and end of the range in the list also. 

Q: I defined a holiday but it's not adjusting the service at all but it does show in green on the calendar. Any suggestions?
A: Check if the holiday handling code is set to "No change to service schedule". If so you'll need to edit the handling code for that holiday entry. To edit all you need to do is tap the holiday entry in the holiday table on the configuration page. Then change the handling code.

Reminders


Q: When is a reminder issued?
A:  A reminder will appear at approximately 00:00 (based on a 24-hour clock) of the reminder day. The reminder day may be set to either "Day before service." or "Day of service.".  So, a reminder is issued a few seconds into the date on which the reminder is to be issued.  The only situation in which this is not the case is when the option "Starting" is set to "Today" and there is a service scheduled for today. More on this below in the item regarding the "Starting" option.

Q: If I change the configuration when will the reminders be updated?
A: The reminders will be updated when the application is closed. For example when you press the home button the reminders are updated. Note that the reminders are only updated when there is a change to any configuration option - not just the reminder options.

Q: What will I see when a reminder is issued?
A: You will see an on-screen reminder, hear a sound and the Trash Tickler application icon on the desktop will be badged with the number 1.

Q: The "Starting" option is a little confusing. Can you explain further?
A: The starting option determines when the first reminder generated will actually be seen. The options are either today or tomorrow. For example, if there is a service shown on the calendar for today and you want the application to generate a reminder for it select the "Today" option. Once you close the application you'll see the alert assuming you also have the "Issued:" option set to "Day of service.". Since reminders are recreated each time the application is closed you'll see this alert as long as there was a change to any configuration option. If you make no subsequent changes then this alert will only be seen once. If you make repeated changes to the configuration and close and reopen the application you'll see this alert multiple times. This might be desirable if testing. If not, then select the "Tomorrow" starting option and reminders will begin the next day instead.

Q: If I move my device between time zones how are reminders handled?
A: Reminders currently do not account for time zone differences. This has typically been sufficient for most people but let us know if this presents a problem and we'll look into it further.

Q: Can I turn off the reminders or make other reminder modifications?
A: Yes. This is done outside of Trash Tickler. If open, close Trash Tickler and then locate the Settings app for your device. Start Settings (the Setting app icon is a set of gears) and then locate the  "Notifications" section. Select "Notifications" and then find Trash Tickler in the list of Apps under "Notification Center". Tap it. The view presented will allow you to make various changes such as turning off notification center (for Trash Tickler only) or changing the alert style.


Help


 Q: Where is the help in the App located?
A: All pages have a small information icon (or two) generally located at or near the bottom right-hand corner of the page. Tap this icon and help will appear for the currently displayed page. As needed the help will be more detailed in some areas and more casual in others.

Q: Once the help is displayed for a page how do I clear it?
A: Tap the information icon again or tap the screen if a page curl is presented.
